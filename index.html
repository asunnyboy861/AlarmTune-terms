2. Description of Service

AlarmTune is an iOS alarm clock application that allows you to set custom volume levels for each alarm independently from your device's system volume. The App provides features including but not limited to:

  • Independent volume control per alarm
  • Volume presets (Whisper, Gentle, Moderate, Loud, Maximum)
  • Gradual fade-in alarm volume
  • 30+ built-in alarm sounds across 5 categories
  • Apple Music library integration (select songs as alarm sounds)
  • Custom audio file import from Files app
  • Video background playback during alarm ringing
  • AI-generated alarm sounds (device-side synthesis)
  • Sound shuffle (automatic daily/weekly sound rotation)
  • Alarm categories and labels
  • Snooze functionality
  • Optional Premium subscription for unlimited imports and AI features
  • Contact Support feedback feature
